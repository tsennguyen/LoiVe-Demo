# Lối Về — Song May Home Demo

> Ứng dụng nhập môn & đặt lịch trải nghiệm 49h tại Sông Mây Home.

## 🌿 Giới thiệu

**LỐI VỀ** là ứng dụng web demo mobile-first, giúp khách hàng:

- Khám phá hành trình 49h nghỉ dưỡng & thiền định
- Xem lịch các đợt sắp tới
- Đăng ký tham gia / đăng ký chờ
- Chuẩn bị hành lý với checklist
- Xem FAQ & thông tin liên hệ

Dành cho khách quét QR code tại quầy trưng bày.

## 📱 Tính năng

| Trang | Mô tả |
|-------|-------|
| Home | Hero section + CTA |
| Onboarding | Khảo sát nhu cầu (chip selection) |
| Guide | Timeline hành trình 3 ngày |
| Schedule | Danh sách đợt + filter |
| Booking | Form đăng ký |
| Waitlist | Đăng ký chờ khi hết chỗ |
| Checklist | Chuẩn bị đồ dùng |
| FAQ | Câu hỏi thường gặp |
| Admin | Dashboard cho quản lý |

### Dark / Light Mode

Toggle ☀️/🌙 góc phải — lưu vào localStorage.

## 🛠️ Công nghệ

- **Next.js 16** (App Router)
- **Vanilla CSS** (CSS Custom Properties cho theming)
- **React 19**

## 🚀 Cài đặt & Chạy

```bash
# Clone
git clone https://github.com/tsennguyen/LoiVe-Demo.git
cd LoiVe-Demo

# Cài dependencies
npm install

# Chạy dev server
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000).

## 📦 Deploy

```bash
npx -y vercel --prod
```
